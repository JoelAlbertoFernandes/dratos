# DRATOS Landing Page V3

Demo comercial narrativa da DRATOS, criada para apresentar a jornada e cada módulo da plataforma com estética enterprise futurista.

## Abrir localmente

Abra `index.html` diretamente no navegador. Não é necessário instalar dependências.

Para testar com servidor local:

```bash
npx serve .
```

## Publicar no GitHub Pages

1. Envie o conteúdo desta pasta para um repositório.
2. No GitHub, acesse `Settings > Pages`.
3. Selecione a branch e a pasta raiz.
4. Salve e aguarde a publicação.

## Estrutura

- `content/`: textos editáveis em PT, EN e ES.
- `css/style.css`: identidade visual e responsividade.
- `js/`: carregamento de conteúdo e interações.
- `assets/`: logos, vídeo, imagens e documentos.
- `COMO_EDITAR.md`: guia prático de conteúdo.
- `PROJETO_INFO.md`: arquitetura e decisões do projeto.
